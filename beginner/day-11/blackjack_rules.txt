Simplified Black Jack rules

Our Blackjack Game House Rules
The deck is unlimited in size.
There are no jokers.
The Jack/Queen/King all count as 10.
The Ace can count as 11 or 1.
Use the following list as the deck of cards:
cards = [11, 2, 3, 4, 5, 6, 7, 8, 9, 10, 10, 10, 10]

The cards in the list have equal probability of being drawn.
Cards are not removed from the deck as they are drawn.
The computer is the dealer.

Dealer and user draw 1 card each, both values shown
Dealers next card is hidden.

if user cards above 21, User loses
if dealer cards above 21, dealer loses
if dealer cards == user cards then draw
if dealers cards == 21, User loses
if Users cards = 21, User wins

If dealers cards are under 17, then dealer draws another card

if users hand is greater than dealer and under 21, user wins
if dealers hand is greater than user and under 21, user loses
